# HostelConnect

Quick entry point for the repository.

## Folder Layout

- `backend/` - Flask API, database schema, and Python scripts
- `frontend/` - React + Vite web application

## Run Locally

### Backend
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
